# 修复标签组功能问题

## 1. 修复导航栏滑动范围问题

**问题**：标签组会被右侧的添加标签和展开视图按钮挡住，无法完全滑出。

**解决方案**：调整布局，确保 `LazyRow` 有足够的宽度，不被右侧按钮遮挡。

**修改点**：
- `TagGroupNavigationBar.kt`：调整 `LazyRow` 的宽度和 padding，为右侧按钮预留空间

## 2. 实现双击编辑功能

**问题**：快速双击标签组无法进入编辑页面。

**解决方案**：实现双击打开编辑页面的功能，包括创建编辑对话框和相关逻辑。

**修改点**：
- `TagGroupNavigationBar.kt`：添加标签组编辑对话框
- 实现双击打开编辑对话框的逻辑
- 实现编辑标签组名称和标签的功能

## 3. 完善展开视图功能

**问题**：展开的附加层只显示"标签组管理"，没有平铺显示所有标签组。

**解决方案**：在展开的附加层中添加所有标签组的列表显示。

**修改点**：
- `TagGroupNavigationBar.kt`：修改展开视图，显示所有标签组
- 使用 `LazyRow` 或 `LazyColumn` 平铺显示标签组
- 添加标签组的选择和管理功能

## 4. 实现标签组与标签的关联管理

**问题**：编辑页面中需要能够修改标签组包含的标签。

**解决方案**：实现标签组与标签的关联管理功能。

**修改点**：
- `TagGroupNavigationBar.kt`：在编辑对话框中添加标签选择功能
- 实现添加/移除标签组中标签的逻辑
